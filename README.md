📝 Task Management Application

A modern and responsive Task Management Application built using HTML, CSS, and JavaScript. This application helps users organize daily tasks with an intuitive interface, persistent local storage, and a built-in dark/light mode.

🚀 Live Demo

🔗 Live Website:
https://abishek-stack.github.io/to-do-list/

---

✨ Features

- ➕ Add new tasks
- ⌨️ Add tasks using the Enter key
- ☑️ Mark tasks as completed
- 🗑️ Delete tasks
- 🌙 Dark / ☀️ Light Mode
- 💾 Local Storage support (tasks persist after page refresh)
- 🚫 Prevents empty task submission
- 📭 Displays a "No tasks yet" message when the task list is empty
- 📱 Responsive and modern user interface

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
